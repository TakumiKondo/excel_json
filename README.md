# Sample of Excel to JSON or JSON to Excel
 
ExcelファイルのデータをJSONに変換する。\
また、変換されたJSONに何らかの処理後、新しいExcelファイルに書き出すサンプルです。
  
# Requirement
 
* node.js 12.18.4
* xlsx 0.16.7
 
# Installation
 
node.js \
https://nodejs.org/ja/
 
# Usage
 
```bash
git clone git@github.com:TakumiKondo/excel_json.git
cd excel_json
npm install
node index.js
```
